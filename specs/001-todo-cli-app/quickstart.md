# Quickstart: Todo CLI App

## Setup

1. **Install uv**:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```

2. **Run the application**:
   ```bash
   uv run src/main.py
   ```

## Development

1. **Install dependencies**:
   ```bash
   uv sync
   ```

2. **Run tests**:
   ```bash
   uv run pytest
   ```
